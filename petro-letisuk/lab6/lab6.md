## Мета роботи  
Ознайомитися з різними типами змінних в об’єктно-орієнтованому програмуванні.

---

## Хід роботи  

1. Створено файл `main.py`, у якому реалізовано клас `Course`.  
   Клас приймає такі аргументи: `course_name`, `instructor`, `duration_in_weeks`.  
   У конструкторі `__init__` ініціалізуються атрибути об’єкта, а також формується додатковий атрибут `course_description`, створений на основі інших властивостей.
2. Реалізовано метод `generate_description()`, який повертає текстовий опис курсу відповідно до значень атрибутів об’єкта.
3. Створено три об’єкти класу: `course1`, `course2`, `course3`, кожен із власними параметрами.  
   Для кожного об’єкта викликано метод `generate_description()`.
4. Реалізовано змінну класу `object_count`, яка відстежує кількість створених об’єктів класу `Course`.
5. Створено метод класу `get_object_count()`, який повертає значення змінної класу.  
   Метод було використано для визначення кількості створених об’єктів.
