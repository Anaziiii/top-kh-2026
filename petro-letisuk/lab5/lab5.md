## Хід роботи  

У ході виконання лабораторної роботи було створено клас `Vehicle`, який описує транспортний засіб.  
Клас містить такі атрибути:
- `name` — назва транспортного засобу  
- `kind` — тип  
- `color` — колір  
- `value` — вартість  

Було реалізовано метод `description(self)`, який повертає текстовий опис об'єкта на основі його властивостей.  

Створено два об'єкти класу — `car1` та `car2`, для яких задано різні значення атрибутів.  

Після цього викликано метод `description()` для кожного об'єкта, що дозволило отримати повний опис транспортних засобів.
