# Лабораторна робота №1

## Хід роботи

1. В процесі роботи було встановлено середовище розробки **Visual Studio Code** на операційну систему Windows 11.
2. Було створено файл `lab1.py` та написано програму мовою Python.
3. Програма реалізує функцію `pow(base, exponent)`, яка обчислює степінь числа:
   - Якщо `exponent == 0` — функція повертає `1`.
   - Якщо `exponent < 0` — повертається обернене значення числа у відповідному степені.
   - Якщо `exponent > 0` — число перемножується саме на себе задану кількість разів за допомогою циклу.
4. Проведено тестування програми. Отримані результати:
   - `pow(2, 3)` - `8`
   - `pow(5, -2)` - `0.04`
   - `pow(3, 0)` - `1`
5. Клоновано Git-репозиторій за допомогою **Git Bash** та завантажено виконану роботу на перевірку.
