**Мета роботи** - ознайомитися з різними типами методів в об’єктно-орієнтованому програмуванні.

## Хід роботи

1. Для виконання даної роботи [lab7](lab7.py), за основу взято роботу №6. Реалізовано два методи класу **new_car** для додавання нового авто, **change_max_speed** для зміни максимальної швидкості.
2. Також додано один статичний метод **km_to_melis**, який конвертує пробіг із кілометрів у милі.

## Висновок
Під час виконання лабораторної роботи було замсвоєно роботу з методами класу а також статичними методами.
