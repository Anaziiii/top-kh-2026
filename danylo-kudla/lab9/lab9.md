## Хід роботи 

1. За основу виконання [lab9](lab9.py) взято клас **Dog** з lab5.
2. Додано метод **__repr__** для налагодження або дебагу використовується програмістами, щоб скопіювавши його можна було зразу створити новий об'єкт.
3. Створено метод **__str__** за допомогою, якого виводиться повна інформація про собаку для звичайного користувача у стрічці.
4. Перевизначено методи **__gt__** і **__add__**, а саме для **__gt__** задано порівняння об'єктів класу. Для **__add__** організовано створення імені з двох об'єктів.

## Висновок
У підсумку виконання лабораторнї роботи навчився працювати з спеціальними методами в Python. Реалізовано метод **__repr__** для отримання технічного опису об'єкта та метод **__str__** для виводу інформації користувачу у рядковому форматі.

Перевизначено методи **__gt__** та **__add__**, що дало можливість напряму взаємодіяти зі стандартними операторами **>** та **+** з об'єктами класу. Це забезпечило чистіший код та інкапсуляцію логіки порівняння й додавання всередині самого класу.
