**Мета роботи** - освоїти роботу з декораторами в Python 3.

## Хід роботи

1. За основу виконання [lab10](lab10.py) взято клас **Dog** та створено клас **Collar**
2. У класі **Collar** створено два атрибути екземпляра, а саме: **self.owner_name** для імені власника та **self.phone_number** для номера телефону.
3. Для класу **Dog** створено атрибут екземпляра **self._chip**, який приймає композицію від **Collar**. Присвоєно метод **chip** для якого реалізується декоратор **@property**, щоб вивести наявну інформацію наче він є атрибутом класу. Далі реалізується декоратор **@chip.setter** в якому можна замінювати попереднє значення імені власника чи номеру телефону.
Щоб видалити дані з нашийника використовується декоратор **@chip.deleter** він видаляє всі значення, що були вказані до його виклику. 
## Висновок
У підсумку лабораторної роботи реалізовано два класи **Collar** та **Dog** із механізмом композиції. Використано декоратор **@property** для того, щоб перетворити атрибут на властивість. Декоратор **@chip.setter** надав можливість модифікації значень атрибутів вже існуючого об'єкта класу не ламаючи логіки попередньо створеної моделі. **@chip.deleter** забезпечив процес очищення даних шляхом видалення або занулення. Результат роботи програми показав успішне виданення та замінення об'єктів.
