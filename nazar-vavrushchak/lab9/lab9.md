## Хід роботи

1. **Поліморфізм та перевизначення методів:** Засвоєно концепцію поліморфізму на практиці. У базовому класі `BasketballPlayer` створено метод `get_role_description()`, який було успішно перевизначено у дочірніх класах `Captain` та `Rookie` для надання їм унікальної поведінки. Продемонстровано поліморфізм у дії: через єдиний цикл список різних об'єктів оброблявся одним викликом методу, але кожен об'єкт реагував по-своєму.

2. **Реалізація магічних методів:** Вивчено та імплементовано спеціальні ("магічні") методи Python, що дозволяють об'єктам взаємодіяти з вбудованими операторами. Для базового класу реалізовано:
   * `__str__` — для форматованого текстового відображення об'єкта під час виклику `print()`.
   * `__add__` — для перевантаження оператора додавання `+` (реалізовано підсумовування загальної кількості очок двох гравців).
   * `__gt__` — для перевантаження оператора порівняння `>` (визначення гравця з вищою результативністю).
